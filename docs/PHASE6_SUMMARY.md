## 🎯 リリース前に必要な作業（ユーザー側）

### 重要度: 🔴 高（必須）

1. **アカウント登録**
   - [ ] Apple Developer Program（iOS） - 年間12,980円
   - [ ] Google Play Console（Android） - 登録$25
   - [ ] Google AdMob アカウント

2. **画像素材の作成**
   - [ ] アプリアイコン（1024x1024px）
   - [ ] スプラッシュ画面（2732x2732px）
   - [ ] スクリーンショット（各サイズ）

3. **Web公開**
   - [ ] プライバシーポリシーをWebに公開
   - [ ] 利用規約をWebに公開
   - [ ] サポートページの作成（オプション）

4. **AdMob設定**
   - [ ] AdMobでアプリ登録
   - [ ] iOS App ID取得
   - [ ] Android App ID取得
   - [ ] Ad Unit ID取得
   - [ ] コードに設定

5. **IAP商品登録**
   - [ ] App Store Connectで商品登録（`remove_ads_pro` / ¥370）
   - [ ] Google Play Consoleで商品登録（`remove_ads_pro` / ¥370）

6. **コード更新**
   - [ ] Bundle ID / Package Name の設定
   - [ ] メールアドレスの設定（プライバシーポリシー・利用規約）
   - [ ] 著作権表示の更新

### 重要度: 🟡 中（推奨）

7. **ビルド・テスト**
   - [ ] ネイティブビルド実行
   - [ ] 実機でテスト
   - [ ] AdMobテスト
   - [ ] IAPテスト（サンドボックス）

8. **最終確認**
   - [ ] クラッシュしないか
   - [ ] すべてのレベルがプレイ可能か
   - [ ] ヒント機能が動作するか
   - [ ] データ永続化が機能するか

## 🚀 リリースまでの流れ

```
1. 画像素材作成（アイコン・スプラッシュ・スクリーンショット）
   ↓
2. プライバシーポリシー・利用規約をWeb公開
   ↓
3. AdMob・IAP設定（App ID・商品登録）
   ↓
4. コード更新（Bundle ID・メールアドレス等）
   ↓
5. ネイティブビルド・テスト
   ↓
6. EAS Buildでストア用ビルド作成
   ↓
7. App Store Connect / Google Play Consoleで情報入力
   ↓
8. 審査提出
   ↓
9. 審査通過後、リリース！
```
