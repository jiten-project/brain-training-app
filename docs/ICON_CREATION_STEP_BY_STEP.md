# アイコン作成：ステップバイステップガイド

このガイドでは、Canvaを使って記憶力トレーニングアプリのアイコンを作成する手順を説明します。

---

## 🎯 作成するアイコン

1. **アプリアイコン**: 1024x1024px（iOS/Android共通）
2. **適応型アイコン**: 1024x1024px（Android用、透過PNG）

---

## 📱 方法1: Canva（推奨・無料）

### Step 1: Canvaにアクセス

1. Canva (https://www.canva.com/ja_jp/) にアクセス
2. 無料アカウントを作成（Googleアカウントで簡単登録）

### Step 2: カスタムサイズを作成

1. ホーム画面で「デザインを作成」をクリック
2. 「カスタムサイズ」を選択
3. サイズを入力：
   - 幅: **1024** px
   - 高さ: **1024** px
4. 「新しいデザインを作成」をクリック

### Step 3: 背景を設定

#### オプションA: グラデーション背景（推奨）

1. 左メニューから「素材」を選択
2. 検索バーに「グラデーション 紫」と入力
3. 気に入ったグラデーションを選択
4. キャンバス全体に配置

**カスタムグラデーション作成:**
1. 左メニューから「素材」→「図形」
2. 四角形を選択してキャンバス全体に配置
3. 上部の色選択をクリック
4. 「グラデーション」タブを選択
5. 色を設定：
   - 上: `#6200EE`（紫）
   - 下: `#9C27B0`（濃い紫）

#### オプションB: 単色背景

1. 左メニューから「素材」→「図形」
2. 四角形を選択
3. 色を `#6200EE` に設定
4. キャンバス全体に配置

### Step 4: メインアイコンを追加

#### 案1: 脳のアイコン（推奨）

1. 左メニューから「素材」を選択
2. 検索バーに「脳 アイコン」と入力
3. 白いシンプルな脳のアイコンを選択
4. 中央に配置
5. サイズを調整（キャンバスの50-60%程度）
6. 色を白（#FFFFFF）に変更

**パズルピースを追加（オプション）:**
1. 「素材」から「パズル ピース」を検索
2. 黄色（#FFC107）のパズルピースを選択
3. 脳の一部に重ねて配置
4. サイズを調整（脳の20%程度）

#### 案2: 電球アイコン

1. 「素材」から「電球 アイコン」を検索
2. シンプルな電球を選択
3. 色を黄色（#FFC107）に変更
4. 中央に配置
5. サイズを調整（キャンバスの60-70%）

**キラキラを追加:**
1. 「素材」から「キラキラ」を検索
2. 白い星や光のエフェクトを選択
3. 電球の周りに3-4個配置

#### 案3: 絵文字グリッド

1. 「素材」から「四角形」を4つ選択
2. 2x2のグリッドに配置
3. 各四角形に絵文字を追加：
   - テキストボックスを作成
   - 絵文字を入力: 🌸 🐶 🍎 🌈
   - フォントサイズを大きく（200pt程度）
4. 各四角形に異なる明るい色を設定

### Step 5: テキストを追加（オプション）

**アプリ名の頭文字を追加する場合:**

1. 左メニューから「テキスト」を選択
2. 「見出しを追加」をクリック
3. 「記」または「脳」と入力
4. フォント設定：
   - フォント: Noto Sans JP（日本語対応）
   - サイズ: 400-500pt
   - 色: 白（#FFFFFF）
   - 太字: オン
5. 中央に配置

**注意**: テキストは小さく表示されると読みにくいため、アイコンのみの方が推奨

### Step 6: 最終調整

#### バランス確認
1. 要素が中央に配置されているか確認
2. 余白が均等か確認
3. 色のコントラストが十分か確認

#### 小サイズでの確認
1. ブラウザのズームを25%に設定
2. アイコンが認識できるか確認
3. 必要に応じてサイズを調整

### Step 7: ダウンロード

1. 右上の「共有」→「ダウンロード」をクリック
2. ファイルの種類: **PNG**
3. サイズ: **推奨サイズ**（1024x1024px）
4. 背景: **透過なし**（重要！）
5. 「ダウンロード」をクリック

### Step 8: ファイルを保存

ダウンロードしたファイルを以下のように保存：

```
brain-training-app/assets/icon.png
```

## 🎨 Android用適応型アイコンの作成

### Step 1: 前景レイヤーを作成

1. Canvaで新しい1024x1024pxデザインを作成
2. **背景は透明に設定**:
   - キャンバスの外側をクリック
   - 「背景色」を選択
   - 「透明」を選択
3. メインアイコン（脳、電球など）のみを配置
4. 周囲に20%のマージンを確保（安全領域）
5. PNG形式でダウンロード（透過背景）

### Step 2: ファイルを保存

```
brain-training-app/assets/adaptive-icon.png
```

## 🎨 方法2: Figma（プロフェッショナル）

Figmaを使いたい場合は、以下の手順：

1. Figma (https://www.figma.com/) にアクセス
2. 無料アカウント作成
3. 新しいファイルを作成
4. フレームを追加（F）
5. サイズを1024x1024に設定
6. デザイン作成
7. Export → PNG → 1x でエクスポート

## 📋 チェックリスト

アイコン作成後、以下を確認：

- [ ] サイズが1024x1024pxである
- [ ] PNG形式で保存されている
- [ ] 背景が透明でない（通常版）
- [ ] 適応型アイコンは透過背景である
- [ ] 小さいサイズ（60x60px）でも認識できる
- [ ] 白背景・黒背景の両方で見やすい
- [ ] `brain-training-app/assets/icon.png` に保存
- [ ] `brain-training-app/assets/adaptive-icon.png` に保存

## 🎨 デザインのヒント

### ✅ 良いアイコンの特徴
- シンプルで認識しやすい
- 色のコントラストが強い
- 小さいサイズでも判別可能
- アプリの機能が一目で分かる

### ❌ 避けるべきこと
- 細かすぎるディテール
- 多すぎる色
- 読みにくい小さなテキスト
- 複雑すぎる構成

## 次のステップ

アイコン作成が完了したら：

1. **app.jsonに設定**:
```json
{
  "expo": {
    "icon": "./assets/icon.png",
    "android": {
      "adaptiveIcon": {
        "foregroundImage": "./assets/adaptive-icon.png",
        "backgroundColor": "#6200EE"
      }
    }
  }
}
```

2. **スプラッシュ画面の作成**: `SPLASH_CREATION_GUIDE.md` を参照

3. **ビルドで確認**:
```bash
npx expo prebuild --clean
```
