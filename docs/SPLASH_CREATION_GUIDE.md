# スプラッシュ画面作成ガイド

このガイドでは、記憶力トレーニングアプリのスプラッシュ画面を作成する手順を説明します。

---

## 🎯 スプラッシュ画面とは

アプリ起動時に表示される最初の画面です。1〜3秒程度表示されます。

### 必要なサイズ
- **2732 x 2732 px**（正方形、推奨）
- PNG形式、透過なし

---

## 🎨 Canvaで作成（推奨）

### Step 1: Canvaでキャンバス作成

1. Canva (https://www.canva.com/ja_jp/) にログイン
2. 「デザインを作成」→「カスタムサイズ」
3. サイズ入力：
   - 幅: **2732** px
   - 高さ: **2732** px
4. 「新しいデザインを作成」をクリック

---

### Step 2: 背景を設定

#### 推奨デザイン: 紫グラデーション

1. 左メニューから「素材」→「図形」
2. 四角形を選択、キャンバス全体に配置
3. 色選択 → 「グラデーション」
4. 色設定：
   - 上: `#6200EE`（紫）
   - 下: `#9C27B0`（濃い紫）
5. グラデーション方向: 上から下

---

### Step 3: アプリアイコンを配置

#### 作成済みのアイコンを使用

1. 左メニューから「アップロード」
2. 作成したアイコン（icon.png）をアップロード
3. キャンバス中央に配置
4. サイズ: **400 x 400 px** 程度

#### アイコン未作成の場合

アイコン作成ガイドに従って先にアイコンを作成してください。

---

### Step 4: アプリ名を追加

#### テキスト配置

1. 左メニューから「テキスト」→「見出しを追加」
2. テキスト入力: **記憶力トレーニング**
3. フォント設定：
   - フォント: **Noto Sans JP**（日本語対応）
   - サイズ: **100-120 pt**
   - 色: **白（#FFFFFF）**
   - 太字: **オン**
   - 位置: **中央揃え**

4. アイコンの下に配置（50-80pxの間隔）

---

### Step 5: サブタイトルを追加（オプション）

1. 「テキスト」→「小見出しを追加」
2. テキスト入力: **脳トレゲーム**
3. フォント設定：
   - フォント: **Noto Sans JP**
   - サイズ: **60-80 pt**
   - 色: **白（#FFFFFF、透明度80%）**
   - 位置: **中央揃え**

4. アプリ名の下に配置（20-30pxの間隔）

---

### Step 6: レイアウト確認

#### 中央配置の確認

1. すべての要素（アイコン、テキスト）を選択
2. 上部メニューから「配置」をクリック
3. 「水平方向中央揃え」を選択
4. 「垂直方向中央揃え」を選択

#### セーフエリアの確認

重要な要素が中央の70%以内に収まっていることを確認

```
┌─────────────────────┐
│      余白 15%       │
│  ┌───────────────┐  │
│  │   [アイコン]  │  │
│  │               │  │
│  │記憶力トレーニング │  │
│  │  脳トレゲーム   │  │
│  │               │  │
│  └───────────────┘  │
│      余白 15%       │
└─────────────────────┘
```

---

### Step 7: ダウンロード

1. 右上の「共有」→「ダウンロード」
2. ファイルの種類: **PNG**
3. サイズ: **推奨サイズ**（2732x2732px）
4. 背景: **透過なし**
5. 「ダウンロード」をクリック

---

### Step 8: ファイルを保存

ダウンロードしたファイルを保存：

```
brain-training-app/assets/splash.png
```

## ⚙️ app.jsonに設定

スプラッシュ画面を作成したら、`app.json`に追加：

```json
{
  "expo": {
    "splash": {
      "image": "./assets/splash.png",
      "resizeMode": "contain",
      "backgroundColor": "#6200EE"
    }
  }
}
```

### resizeModeの選択

- **contain**（推奨）: 画像全体を表示、余白あり
- **cover**: 画面全体をカバー、一部が切れる可能性
- **native**: デバイスごとに最適化

## 🔍 プレビュー方法

### Expoアプリで確認

1. スプラッシュ画面を設定
2. 開発サーバーを再起動
```bash
npm start
```
3. Expoアプリを再起動して確認

### ネイティブビルドで確認

```bash
npx expo prebuild --clean
npx expo run:ios
# または
npx expo run:android
```

## 💡 デザインのヒント

### ✅ 良いスプラッシュ画面
- シンプルで美しい
- ブランドカラーを使用
- 読み込み時間を感じさせない
- アプリの第一印象を良くする

### ❌ 避けるべきこと
- 情報を詰め込みすぎる
- 小さすぎるテキスト
- 複雑すぎるデザイン
- 読み込み中のアニメーション（静止画推奨）

## 🎬 アニメーション（オプション）

Expoでは、以下のライブラリでスプラッシュ画面をカスタマイズ可能：

```bash
npx expo install expo-splash-screen
```

アニメーション例:
- フェードイン
- スライドイン
- ロゴの拡大

詳細: https://docs.expo.dev/develop/user-interface/splash-screen/

---

## 📱 デバイス別の表示

スプラッシュ画面は、デバイスの画面サイズに応じて自動調整されます：

- **iPhone**: 縦長の場合、上下に余白
- **iPad**: ほぼ正方形に近い
- **Android**: 機種により異なる

`resizeMode: "contain"` を使用することで、すべてのデバイスで適切に表示されます。
