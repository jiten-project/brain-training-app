# 絵合わせ記憶力トレーニング

高齢者向け脳トレスマホアプリ

## 📱 概要

記憶力と認識力のトレーニングを目的とした、高齢者に優しいデザインのスマホアプリです。

## 🎮 ゲームの流れ

1. **記憶フェーズ**: 表示された絵を覚える
2. **回答フェーズ**: 選択肢から正解の絵を選ぶ
3. **結果フェーズ**: 正解数と応援メッセージを表示

## 📊 レベル

- **レベル数**: 1〜20
- **画像枚数**: レベル数 + 3枚
  - レベル1: 4枚
  - レベル10: 13枚
  - レベル20: 23枚
- **クリア条件**: 80%以上正解
- **進行方式**: 段階式解放（クリアしたレベルまでプレイ可能）

## ⚙️ ゲームモード

### 初級 ★☆☆ (デフォルト)
- **選択肢数**: 正解枚数 + 10枚
- **例**: レベル5 (8枚) → 18枚から選択
- **対象**: 初めての方、ゆっくり楽しみたい方

### 中級 ★★☆
- **選択肢数**: 正解枚数 × 3倍
- **例**: レベル5 (8枚) → 24枚から選択
- **対象**: 慣れてきた方

### 上級 ★★★
- **選択肢数**:
  - レベル1-10: 固定50種類
  - レベル11-20: 固定100種類
- **例**: レベル15 (18枚) → 100枚から選択
- **対象**: 上級者向け、最高難易度に挑戦したい方

## 🎨 高齢者向け配慮

- **大きな文字**: 最小18pt、重要情報は24pt以上
- **大きなボタン**: 最小60×60pt
- **高コントラスト配色**: 見やすい色使い
- **シンプルな操作**: 直感的なUI
- **視覚的フィードバック**: 選択時のハイライト、チェックマーク

## 💾 データ管理

- **ローカル保存**: 端末内にデータを保存
- **保存内容**:
  - 最高到達レベル
  - クリア済みレベル
  - ゲームモード設定
  - 音声ON/OFF設定
- **リセット機能**: 設定画面から全データ削除可能

## 🛠️ 技術スタック

- **フレームワーク**: React Native (Expo)
- **言語**: TypeScript
- **UI**: React Native Paper
- **ナビゲーション**: React Navigation
- **ストレージ**: AsyncStorage
- **状態管理**: Context API

## 🚀 開発・実行

### インストール

```bash
cd brain-training-app
npm install
```

### 起動

```bash
npm start
```

### iOS実機で実行

```bash
npm run ios
```

### Android実機で実行

```bash
npm run android
```
